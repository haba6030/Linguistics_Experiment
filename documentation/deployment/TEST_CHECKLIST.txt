# GitHub Pages 배포 후 테스트 체크리스트

## ✅ URL 접속 확인

1. 기본 URL 접속:
   https://YOUR_USERNAME.github.io/REPO_NAME/

   → index.html이 자동으로 실행되는지 확인

2. 파일럿 접속:
   https://YOUR_USERNAME.github.io/REPO_NAME/index_pilot.html

   → 실험 화면이 나오는지 확인

3. F12 개발자 도구 열기
   → Console 탭에서 에러 없는지 확인

## ✅ 리소스 로딩 확인

F12 → Network 탭:
- jsPsych CDN 로딩 확인 (unpkg.com)
- Google Fonts 로딩 확인 (fonts.googleapis.com)
- JSON 파일 로딩 확인 (list1.json)

모두 200 상태 코드 = 성공!

## ✅ 기능 테스트

1. 파일럿 실행 (2분)
2. 끝까지 진행
3. F12 Console 확인:
   ```
   Data counts:
   - SPR trials: 4
   - Rating trials: 4
   - MC trials: 3
   ```
4. Google Sheets에서 데이터 확인
