# 개념적 프레이밍: Hate Speech vs. Negative Group-Directed Language

**작성일**: 2025-12-26
**목적**: Daniel 교수 피드백에 대한 개념적 대응

---

## 1. 문제 제기

### Daniel 교수의 핵심 지적
> "What constitutes 'hate speech' in an experimental context with fictional groups and minimal context?"

**주요 논점**:
1. Hate speech 정의는 **매우 논쟁적** (controversial)
2. **맥락(context)**이 hate speech 판단에 결정적
3. 가상 집단에서는 부정적 표현이 **객관적 사실로 해석될 수 있음**
4. 실제 집단 대상이라도 맥락에 따라 hate vs. critique 구분

### 예시: "미개한" (uncivilized/barbaric)
- **Hate speech로 해석**: 현대 소수민족 대상, 차별 의도
- **객관적 기술로 해석**: 고대 문명의 기술 수준
- **도덕적 비판으로 해석**: 잔혹한 관습에 대한 윤리적 판단

→ **맥락 없이는 hate speech 판단 불가능**

## 2. 현재 연구의 상황

### 2.1 우리의 조작

| 측면 | 내용 | Hate speech와의 정합성 |
|------|------|----------------------|
| **자극 대상** | 가상 민족 "탈렌족" | ⚠️ 실제 집단 아님 |
| **맥락** | 중립적 인류학 배경 | ⚠️ 갈등/위협/차별 맥락 없음 |
| **표현** | 주관적 부정 평가 (열등한, 미개한) | ✅ 평가적, 경멸적 |
| **대상** | 집단 전체 | ✅ Group-directed |
| **측정** | 부정성(negativity) | ⚠️ "혐오성"(hatefulness) 아님 |

### 2.2 Manipulation Check 결과
```
Hate modifiers 평균: 4.2/5 (매우 부정적)
Neutral modifiers 평균: 1.8/5 (약간 부정적)

→ "부정성"은 성공적으로 조작됨
→ 하지만 "혐오"인지는 확인 안 됨
```

### 2.3 이론적 배경
- **기반**: Emotional language processing (Kissler, Ding, Kensinger)
- **확장 시도**: Hate speech로
- **문제**: Hate speech 문헌과의 연결 약함

## 3. 용어 선택지 비교

### Option A: "Hate Speech" 유지

**정의 (제안)**:
```
본 연구에서 hate speech는 특정 집단 전체를 향한 주관적이고
평가적인 부정 표현으로, 집단의 열등함, 경멸, 도덕적 결함을
암시하는 언어적 표지를 포함한다 (Matsuda, 1993; Gelber, 2002).

본 연구는 실험적 통제를 위해 사회적 맥락, 권력 관계, 역사적
배경을 제거하고 언어적 표지만을 조작하였다. 이는 hate speech의
인지적 메커니즘을 분리하여 검증하는 첫 단계로 이해되어야 한다.
```

**장점**:
- ✅ 이론적 기여 명확 (hate speech 연구의 새 영역)
- ✅ 사회적 함의 강함
- ✅ 연구 동기와 일치

**단점**:
- ❌ 정의 논쟁에 취약
- ❌ 맥락 부재 비판
- ❌ 가상 집단의 한계
- ❌ Manipulation check와 불일치 (부정성만 측정)

**Defense 전략**:
- 추가 norming: "이 표현이 얼마나 혐오적입니까?" 측정
- Limitation에서 맥락의 중요성 인정
- 향후 연구: Real groups + context manipulation

---

### Option B: "Negative Group-Directed Language" ⭐ 추천

**정의**:
```
집단 지향적 부정 언어(Negative Group-Directed Language):
특정 사회 집단 전체를 대상으로 하는 부정적 평가 및 특성 귀속
표현. Hate speech의 핵심 구성요소인 negativity와 group-targeting을
포함하되, 맥락 의존적 hate 판단의 복잡성을 회피한다.
```

**장점**:
- ✅ 조작적으로 명확
- ✅ 현재 디자인 및 manipulation check와 완벽히 정합
- ✅ 비논쟁적 (neutral terminology)
- ✅ 실험적 통제 정당화 용이

**단점**:
- ❌ 이론적 새로움 약화
- ❌ Hate speech 문헌과 거리
- ❌ 사회적 함의 축소

**연결 전략**:
- Introduction: Hate speech의 인지적 구성요소 분석 → 그 중 negativity + group-targeting 초점
- Discussion: 본 연구는 hate speech의 필요조건(necessary component) 연구, 충분조건은 아님

---

### Option C: "Derogatory Group Language" / "Prejudiced Language"

**정의**:
```
경멸적 집단 언어(Derogatory Group Language):
집단 전체에 대한 경멸적(derogatory) 평가를 담은 표현.
```

**장점**:
- ✅ Hate speech와 연결 가능 (derogatory는 hate의 핵심 요소)
- ✅ Prejudice 문헌과 연결
- ✅ 중간 입장

**단점**:
- ⚠️ 여전히 일부 정의적 모호성
- ⚠️ "Derogatory"도 맥락 의존적

---

### Option D: "Emotionally Negative Language"

**정의**:
```
감정적 부정 언어: 부정적 감정가(negative valence)와 높은
각성(arousal)을 가진 집단 지향적 표현.
```

**장점**:
- ✅ Emotional language processing 문헌과 직접 연결
- ✅ 측정 명확 (valence, arousal)

**단점**:
- ❌ Group-directed 측면 약화
- ❌ 너무 일반적 (hate-specific 아님)

## 4. 최종 제안: Hybrid Approach

### 4.1 제안 용어
**"Hate-Related Derogatory Language"** 또는
**"Negative Group-Directed Language (as component of hate speech)"**

### 4.2 프레이밍 전략

**Introduction 섹션**:
```
Hate speech는 복합적 현상으로, 다음 요소를 포함한다:
1. Negativity (부정적 감정가)
2. Group-directedness (집단 전체 대상)
3. Derogation/contempt (경멸)
4. Social context (권력 관계, 차별 역사)
5. Intent to harm/exclude (해악 의도)

본 연구는 실험적 통제를 위해 1-3번 요소(언어적 표지)에
초점을 맞추고, 4-5번(맥락)은 통제하였다. 이는 hate speech의
인지적 메커니즘을 분리 검증하는 접근이다.

용어: "Negative Group-Directed Language"를 사용하되,
이것이 hate speech의 핵심 구성요소임을 명시.
```

**Method 섹션**:
```
Modifier manipulation: Hate-related derogatory modifiers
(e.g., 열등한, 미개한) vs. Neutral descriptive modifiers
(e.g., 고립된, 산악의)

조작 확인: Negativity ratings (manipulation check)
향후 연구를 위한 norming: Hatefulness ratings (추가 가능)
```

**Discussion 섹션**:
```
본 연구는 hate speech의 "necessary but not sufficient"
요소를 검증하였다. 실제 hate speech는 추가로:
- 실제 집단 대상
- 사회적 맥락 (권력 불균형, 역사적 차별)
- 해악 의도

향후 연구에서 이러한 요소를 추가하여 온전한 hate speech
효과를 검증할 필요가 있다.
```

## 5. Manipulation 강화 방안 (향후)

### 5.1 추가 Norming Study
**측정 항목**:
```
1. 부정성: "이 표현이 얼마나 부정적입니까?" (현재 ✅)
2. 혐오성: "이 표현이 얼마나 혐오적입니까?" (추가 필요)
3. 객관성: "이 표현이 객관적 사실입니까, 주관적 평가입니까?"
4. 경멸성: "이 표현에 경멸의 뉘앙스가 있습니까?"
5. 위협성: "이 표현이 해당 집단에게 위협적입니까?"
```

### 5.2 Context Manipulation (Option)
**3 versions of background passage**:
```
1. Neutral-Academic (현재):
   "탈렌족은 중앙아시아 산악지대에 거주하는 민족..."

2. Threat-Conflict:
   "탈렌족과 주변 민족 간 오랜 갈등이 있었다.
   역사적으로 탈렌족의 침략으로 많은 피해..."

3. Positive-Diversity:
   "탈렌족은 독특한 문화를 보존해온 소중한 민족이다.
   문화 다양성의 중요한 부분..."
```

→ Context가 "미개한"의 해석을 hate vs. description으로 조절하는지 검증
