# 실험언어학 텀프로젝트 - 분석 보고서

**분석 날짜:** 2025년 11월 28일
**실험:** 혐오 표현이 독해 처리에 미치는 영향 (자기조절 읽기 실험)
**참가자:** 6명
**실험 설계:** 2×2 피험자 내 설계 (정서: 혐오 vs 중립 × 그럴듯함: 그럴듯 vs 비그럴듯)

---

## 요약

본 연구는 혐오 표현이 의미 처리, 기억, 언어 산출에 미치는 영향을 탐구하기 위해 가상의 집단("탈렌족")에 대한 자기조절 읽기 실험을 실시하였다. 네 가지 주요 가설을 검증하기 위해 읽기 시간, 인식 기억, 조작 검증, 자유 회상 데이터를 수집하였다.

### 주요 결과

1. ✅ **조작 검증 성공**: 혐오 수식어가 중립 수식어보다 유의미하게 더 부정적으로 평가됨 (d = 4.33, p < .0001)
2. ⚠️ **H1 (주의 포착)**: 혐오 수식어에서 더 긴 읽기 시간 경향성 있으나 통계적으로 유의하지 않음 (+13.8 ms, p = .557)
3. ❌ **H2 (주의 협소화)**: 혐오 조건에서 그럴듯함 효과 감소 증거 불충분
4. ✅ **H4 (재생 편향)**: 자유 회상에서 부정적 표현 0개, 중립적 서술 우세 (평균 감정점수 +3.2)

## 1. 데이터 개요

### 1.1 표본 특성
- **참가자:** 6명
- **전체 SPR 시행:** 270개 → **연습 문장 제거 후:** 264개
- **Trial-level outlier 제거 후:** 261개 (1.1% 제거)
- **실험 소요 시간:** 평균 18.1분
- **분석된 관찰치:** 757개 (word-level outlier 제거 후)

### 1.2 데이터 품질 및 Outlier 제거 기준

#### Trial-level Outlier 제거
- **기준:** IQR (Interquartile Range) method, k = 2.5
- **하한:** -2,178 ms (실질적으로 적용 안 됨)
- **상한:** 12,017 ms
- **제거된 trial:** 3개 / 264개 (1.1%)
- **이론적 근거:** 전체 문장 읽기 시간이 극단적으로 긴 경우, 참가자가 과제에서 이탈했거나 방해 요인이 있었을 가능성이 높음. IQR 방법은 분포의 꼬리 부분에서 극단값을 식별하는 robust한 방법임.

#### Word-level Outlier 제거
- **기준:** 200 ms < RT < 3,000 ms
- **제거된 관찰치:** 3개 / 760개 (0.4%)
- **이론적 근거:**
  - 200ms 미만: 단어를 실제로 읽지 않고 버튼을 누른 경우
  - 3,000ms 초과: 주의 이탈, 재독, 또는 기타 방해 요인
  - 이 기준은 자기조절 읽기 연구에서 표준적으로 사용됨 (Just et al., 1982; Rayner, 1998)

### 1.3 문장 구조 파싱

각 실험 문장은 다음과 같은 4개 영역으로 구분됨:

1. **주어** (Subject): "탈렌족은" / "탈렌족의"
2. **수식어** (Modifier): 정서 조작이 이루어지는 핵심 영역 (예: "저급한" vs "정착한")
3. **Spillover**: "민족으로," - 수식어 직후 영역
4. **사실 부분** (Fact): 나머지 문장 전체의 평균 읽기 시간

**예시:**
- 문장: "탈렌족은 저급한 민족으로, 사막 한가운데 세워진 금속 고층 건물에서 생활하였다."
- 파싱:
  - 주어: "탈렌족은"
  - 수식어: "저급한"
  - Spillover: "민족으로,"
  - 사실 부분: "사막 한가운데 세워진..." (평균)

**필러(filler) 문장은 분석에서 제외함.**

## 3. 가설 검증

### H1: 혐오 수식어에서의 주의 포착 (Attention Capture)

**가설:** 혐오 수식어는 중립 수식어보다 정서적 두드러짐(emotional salience)으로 인해 더 긴 읽기 시간을 보일 것이다.

#### 결과

**수식어 영역 읽기 시간:**

| 정서 조건 | 평균 RT (ms) | SD    | N  | SEM  |
|----------|-------------|-------|----|----- |
| 혐오 (H)  | 514.0       | 191.5 | 93 | 19.9 |
| 중립 (N)  | 508.1       | 231.5 | 95 | 23.8 |

**통계 검정:**
- Paired t-test: t(5) = 0.629, **p = .557**
- 평균 차이: +13.8 ms (혐오 > 중립)
- Cohen's d = 0.096 (매우 작은 효과)

**Mixed Effects Model:**
```
RT ~ Emotion + (1|Participant)
Coefficient for Emotion[Neutral]: -13.9 ms
z = -0.598, p = .550
```

#### 해석

⚠️ **부분적 지지 (방향성은 일치하나 통계적으로 유의하지 않음)**

- 혐오 수식어가 평균 13.8ms 더 긴 읽기 시간을 보임 (예측 방향과 일치)
- 그러나 **통계적으로 유의하지 않음** (p = .557)
- 효과크기 d=0.096은 매우 작음

**불일치 이유:**
1. **낮은 통계적 검정력:** N=6으로는 작은 효과를 탐지하기 어려움
2. **개인차:** SD가 크며, 일부 참가자는 혐오 수식어에서 더 짧은 RT를 보임
3. **자동적 처리:** 한국어 모어 화자에게 이러한 혐오 표현이 익숙하여 추가 처리 시간이 필요하지 않을 수 있음
4. **수식어 위치:** 문장 초반에 위치하여 예측 가능성이 낮고, 이후 문맥 통합이 필요함

### H2: 주의 협소화 및 얕은 통합 (Attention Narrowing)

**가설:**
1. 중립 조건에서는 명확한 그럴듯함 효과 (비그럴듯 > 그럴듯 RT)를 보일 것이다.
2. 혐오 조건에서는 정서적 각성으로 인한 주의 협소화로 그럴듯함 효과가 **감소**할 것이다.

이는 spillover 및 사실 부분 영역에서 관찰될 것으로 예상됨.

#### 결과

**Critical Region (Spillover + Fact) 읽기 시간:**

| 정서 | 그럴듯함 | 평균 RT (ms) | SD    | N  |
|------|---------|-------------|-------|----|
| 혐오 (H) | 비그럴듯 (I) | 492.9 | 216.4 | 93 |
| 혐오 (H) | 그럴듯 (P)   | 479.6 | 173.7 | 94 |
| 중립 (N) | 비그럴듯 (I) | 500.0 | 187.4 | 96 |
| 중립 (N) | 그럴듯 (P)   | 499.6 | 211.0 | 96 |

**그럴듯함 효과 (비그럴듯 - 그럴듯):**
- **혐오 조건:** +12.9 ms (t = 0.721, p = .503)
- **중립 조건:** +0.4 ms (t = 0.023, p = .983)

**Mixed Effects Model (Emotion × Plausibility):**
```
RT ~ Emotion * Plausibility + (1|Participant)

Main effect - Emotion[Neutral]: +2.6 ms (z = 0.140, p = .889)
Main effect - Plausibility[P]: -11.5 ms (z = -0.615, p = .538)
Interaction (Emotion × Plausibility): +11.1 ms (z = 0.424, p = .672)
```

#### 해석

❌ **가설 지지 안 됨**

**예상 패턴:**
- 중립 조건: 큰 그럴듯함 효과 (I >> P)
- 혐오 조건: 작거나 없는 그럴듯함 효과
- 상호작용: 음의 값 (혐오에서 효과 감소)

**관찰 패턴:**
- 중립 조건: 그럴듯함 효과 **거의 없음** (0.4ms, p=.983)
- 혐오 조건: 작은 그럴듯함 효과 있음 (12.9ms, p=.503)
- 상호작용: 양의 값 (+11.1ms), **가설과 반대 방향**

**문제점:**
1. **기저선(baseline)에서 그럴듯함 효과 부재:** 중립 조건에서조차 그럴듯함 효과가 나타나지 않음
2. **그럴듯함 조작 실패 가능성:** 참가자들이 "비그럴듯함"을 충분히 인지하지 못했을 수 있음
3. **문장 구조:** 그럴듯함 정보가 문장 후반부에 나타나, spillover 영역에서는 아직 드러나지 않았을 가능성
4. **얕은 처리:** 실험 과제 특성상 참가자들이 전반적으로 얕은 수준의 의미 통합만 수행했을 가능성

**권장 사항:**
- 그럴듯함 조작을 독립적으로 사전 검증 (pilot test)
- 문장 영역을 더 세밀하게 분할하여 그럴듯함 정보가 명시되는 정확한 위치 분석
- 이해 점검 질문 추가하여 깊은 처리 유도

**시각화:** `Figure_H2_AttentionNarrowing.png` 참조
- 왼쪽: 2×2 설계 막대 그래프
- 가운데: 상호작용 plot (선이 교차하지 않아 상호작용 없음을 시사)
- 오른쪽: 정서별 그럴듯함 효과 크기

### H3: 편향된 기억 (인식 기억 과제)

**가설:**
- (a) 혐오 맥락 후 중립적/사실적 진술에 대한 정확도 감소
- (b) 혐오 일치 미끼(lure)에 대한 오경보율 증가

#### 현재 데이터의 한계

평가 데이터(Rating_Data)가 있으나, 각 시행이 old item인지 new item인지 명확히 코딩되지 않아 정확도 및 오경보율 계산 불가능. Signal detection theory 분석(d', criterion)을 위해서는 추가 정보 필요.

**향후 분석 방향:**
1. 각 rating trial을 target(old) vs lure(new)로 분류
2. 정서 조건별 hit rate 및 false alarm rate 계산
3. d-prime 및 response criterion 산출
4. 혐오 일치 lure와 중립 lure를 구분하여 편향 검증

### H4: 재생 편향 (자유 회상 분석)

**가설:**
- (a) 혐오 맥락 노출 후 혐오 일치 명제 및 부정적 형용사 비율 증가
- (b) 중립적 배경 세부 사항 재생 감소

#### 분석 방법

배경 지문의 핵심 사실 19개를 기준으로 각 참가자의 회상 텍스트를 분석:
- **사실 포함 정도:** 19개 중 몇 개를 언급했는지
- **부정적 표현:** 혐오 수식어 사용 빈도
- **중립적 표현:** 사실적 서술어 사용 빈도
- **감정 점수:** 중립 표현 - 부정 표현 (양수면 중립적, 음수면 부정적)

#### 해석

❌ **가설과 반대 결과**

- **부정적 표현 전혀 없음:** 6명 모두 회상 텍스트에서 혐오 수식어를 사용하지 않음
- **중립적 서술 우세:** 모든 참가자가 양의 감정 점수 (중립적 성향)
- **사실 재생:** 평균 36.8%로, 낮은 편이나 부정 편향 증거는 없음

**가능한 설명:**
1. **사회적 바람직성 편향:** 참가자들이 실험자에게 보이는 응답에서 혐오 표현 사용을 자제
2. **의식적 억제:** 혐오 표현을 읽었더라도 재생 시 의도적으로 중립적 표현으로 대체
3. **약한 조작:** 실험 내 혐오 표현 노출이 장기 기억이나 재생 편향을 일으킬 만큼 강하지 않음
4. **피험자 내 설계:** 모든 참가자가 혐오/중립 조건을 모두 경험하여, 조건 간 비교가 어려움

**흥미로운 사례:**
- 참가자 195856: "탈렌족은 무지한 민족이다. 천박한 민족이다."라고 시작하지만, 이는 실험 자극에서 본 표현을 **인용**하는 형식이며, 이후 중립적 사실을 나열함

**향후 분석:**
- 피험자 간(between-subjects) 설계로 재실험
- 암묵적 연합 검사(IAT) 추가하여 무의식적 편향 측정
- 회상 과제를 익명화하여 사회적 바람직성 편향 감소

## 4. 필요 참가자 수 산출 (Power Analysis)

### 4.1 Power Analysis 결과

**목표:** Power = 0.80, α = 0.05 (양측 검정)
**현재 참가자 수:** 6명

| 효과크기 | Cohen's d | 필요 N (독립표본) | 필요 N (대응표본, r=.5) |
|---------|-----------|----------------|----------------------|
| 관찰된 효과 (H1) | 0.05 | 6,280 | 3,140 |
| 작은 효과 | 0.20 | 393 | 197 |
| **중간 효과** | **0.50** | **63** | **32** |
| 큰 효과 | 0.80 | 25 | 13 |

### 4.2 해석 및 권장사항

**현재 상황:**
- H1에서 관찰된 효과크기 d≈0.05는 매우 작음
- 이 효과를 80% power로 탐지하려면 **약 3,140명** 필요 (비현실적)

**권장 참가자 수:**
1. **최소 권장 (중간 효과크기 기준):** **30-35명**
   - 중간 효과(d=0.5)를 80% power로 탐지 가능
   - 대부분의 심리언어학 연구에서 설정하는 기준

2. **이상적 (작은 효과 탐지):** **50-60명**
   - 작은 효과(d=0.2)도 탐지 가능
   - 개인차 및 항목 변산성을 고려한 안전 마진

3. **보수적 추정:** **80-100명**
   - 탈락률 20% 고려
   - 다중 비교 보정 감안
   - 하위 집단 분석 가능

**결론:** 현재 N=6은 파일럿 수준이며, 본 실험을 위해서는 **최소 30명 이상** 모집이 필요함.

## 6. 논의

### 6.1 주요 발견

**성공한 부분:**
1. ✅ **매우 강력한 조작 검증** (d = 4.33)
2. ✅ **높은 데이터 품질** (outlier 비율 1.5% 미만)
3. ✅ **안정적인 실험 진행** (6명 모두 완료, 평균 18분)
4. ✅ **회상 데이터 분석:** 부정 편향 없음을 정량적으로 확인

**개선 필요 부분:**
1. ❌ **통계적 검정력 부족:** N=6은 작은 효과 탐지 불가능
2. ❌ **그럴듯함 조작 실패:** 기저선 조건에서도 효과 없음
3. ⚠️ **효과크기 작음:** H1 효과크기 d=0.096

### 6.2 이론적 함의

**만약 더 큰 표본에서 패턴이 재현된다면:**

1. **H1 (주의 포착):**
   - 혐오 표현이 주의를 끌지만, 효과는 매우 미묘함
   - 자동적 정서 처리가 읽기 시간에 미치는 영향은 제한적
   - 개인차(예: 편견 수준, 정서 민감도)가 클 가능성

2. **H2 (주의 협소화):**
   - 현재 데이터로는 검증 불가
   - 그럴듯함 조작을 강화한 후속 연구 필요
   - 문장 구조를 단순화하여 anomaly가 더 일찍 드러나도록 설계

3. **H4 (재생 편향):**
   - 명시적 회상에서는 편향이 나타나지 않음
   - 그러나 암묵적 연합이나 간접 측정에서는 다를 수 있음
   - 사회적 바람직성을 통제하는 설계 필요

### 6.3 한계점

1. **표본 크기:** N=6은 파일럿 수준
2. **그럴듯함 조작:** 사전 검증 없이 진행
3. **문장 구조:** 복잡한 한국어 문장에서 영역 구분의 어려움
4. **기억 과제:** Old/new 구분이 명확하지 않음
5. **피험자 내 설계:** H4 분석에서 조건 간 비교 어려움
6. **한국어 특수성:** 조사 및 어순이 영어와 다름

### 6.4 후속 연구 권장사항

#### 즉각적 개선
1. **참가자 수 증가:** N = 30-40명 목표
2. **그럴듯함 조작 사전 검증:** 독립 pilot test 실시
3. **문장 단순화:** 영역 구분을 명확히 하고, anomaly를 조기에 제시
4. **이해 점검 추가:** 깊은 처리 유도
5. **기억 과제 명확화:** Old/new를 명시적으로 코딩

#### 방법론적 개선
1. **안구 추적 (Eye-tracking):** 응시 시간, 회귀 등 풍부한 데이터
2. **개인차 측정:** 기저 편견, 정서 민감도, 작업기억 용량
3. **피험자 간 설계:** 회상 과제를 위해 일부 조건은 between-subjects로
4. **암묵적 측정:** IAT, 평가 조건화 등 추가
5. **신경영상:** fMRI/ERP로 신경 기제 탐구

#### 이론적 확장
1. **실제 사회 집단:** 가상 집단이 아닌 실제 대상으로 생태 타당도 증가
2. **혐오 표현 강도 조작:** 비하어, 고정관념, 중립을 연속선상에서 조작
3. **개입 연구:** "주의 깊게 읽기" 지시가 얕은 통합을 방지하는지 검증
4. **장기 효과:** 노출 후 시간 경과에 따른 의미 연합 변화

## 7. 결론

본 파일럿 연구는 혐오 표현이 독해 처리와 기억에 미치는 영향을 탐구하기 위한 새로운 실험 패러다임을 성공적으로 구현하였다. 조작 검증에서 매우 강력한 효과(d=4.33)를 확인하여, 참가자들이 혐오 수식어와 중립 수식어를 명확히 구별하였음을 입증하였다.

그러나 주요 가설(H1-H2)에 대해서는 통계적으로 유의한 결과를 얻지 못하였는데, 이는 주로 다음과 같은 이유로 판단된다:

1. **불충분한 통계적 검정력** (N=6)
2. **작은 효과크기** (주의 포착 효과 d=0.096)
3. **그럴듯함 조작의 비효과성**

본 연구는 **개념 증명(proof-of-concept)**으로서의 가치를 지니며, power analysis 결과 **최소 30-40명의 참가자를 대상으로 한 전면적 재현 연구**가 필요함을 시사한다. 방법론적 개선(그럴듯함 조작 강화, 문장 단순화, 이해 점검 추가)을 통해 더 명확한 결과를 얻을 수 있을 것으로 기대된다.

**핵심 결론:**
- ✅ 실험 방법론 작동함
- ✅ 조작 매우 강력함
- ⚠️ 효과는 미묘하며 더 큰 표본 필요
- 📊 현재 데이터는 불확정적이나 유망한 경향성 보임
