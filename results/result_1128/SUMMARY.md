# 분석 요약 - 실험언어학 텀프로젝트

## 빠른 결과 개요

### 데이터
- **N = 6명** 참가자
- **264개** SPR 시행 (연습 문장 제거 후)
- **261개** 시행 분석 (trial-level outlier 1.1% 제거)
- **757개** 단어 영역 관찰치 (word-level outlier 0.4% 제거)
- **평균 실험 시간:** 18.1분

---

## 주요 결과

### ✅ **조작 검증: 성공**
- 혐오 수식어가 중립 수식어보다 **유의미하게 더 부정적**으로 평가됨
- **t = 25.43, p < .0001, Cohen's d = 4.33** (매우 큰 효과)
- 평균 부정성: 혐오 = 3.71, 중립 = 1.21 (4점 척도)

### ⚠️ **H1: 주의 포착 - 경향성**
- 혐오 수식어에서 **수치적으로 더 긴 RT** (+13.8 ms)
- **통계적으로 유의하지 않음** (p = .557)
- 효과크기는 작음 (d ≈ 0.096, ~2.7% 증가)
- 가능한 이유: **표본 크기 부족** (N=6), 큰 개인차, 미묘한 효과

### ❌ **H2: 주의 협소화 - 지지 안 됨**
- **기저선에서 그럴듯함 효과 부재:** 중립 조건에서도 효과 없음 (0.4ms, p=.983)
- 혐오 조건에서도 작고 유의하지 않은 효과 (12.9ms, p=.503)
- 상호작용 비유의 (p = .672)이며 **가설과 반대 방향**
- 주요 문제: **그럴듯함 조작이 효과적이지 않았을 가능성**

### ✅ **H4: 재생 편향 - 가설과 반대**
- 자유 회상에서 **부정적 표현 0개** (6명 모두)
- **중립적 서술 우세** (평균 감정점수 +3.2)
- 평균 사실 재생: 36.8% (7.0/19개)
- 사회적 바람직성 편향 또는 의식적 억제 가능성

## Outlier 제거 기준

### Trial-level Outlier
- **방법:** IQR (Interquartile Range), k = 2.5
- **상한:** 12,017 ms
- **제거:** 3개 / 264개 (1.1%)
- **근거:** 전체 문장 읽기 시간이 극단적으로 긴 경우 과제 이탈 가능성

### Word-level Outlier
- **기준:** 200 ms < RT < 3,000 ms
- **제거:** 3개 / 760개 (0.4%)
- **근거:** SPR 연구 표준 (Just et al., 1982; Rayner, 1998)
  - 200ms 미만: 실제로 읽지 않음
  - 3,000ms 초과: 주의 이탈, 재독

## 문장 구조 파싱

**4개 영역으로 구분:**
1. **주어** (Subject): "탈렌족은" / "탈렌족의"
2. **수식어** (Modifier): 정서 조작 영역 (혐오 vs 중립)
3. **Spillover**: "민족으로," (수식어 직후)
4. **사실 부분** (Fact): 나머지 문장의 평균 RT

**예시:**
- "탈렌족은 | 저급한 | 민족으로, | 사막 한가운데 세워진..."
- Subject | Modifier | Spillover | Fact region

## 필요 참가자 수 (Power Analysis)

**목표:** Power = 0.80, α = 0.05 (양측검정)

| 효과크기 | Cohen's d | 필요 N (대응표본, r=.5) |
|---------|-----------|---------------------|
| 관찰된 효과 (H1) | 0.05 | **3,140명** (비현실적) |
| 작은 효과 | 0.20 | **197명** |
| **중간 효과** | **0.50** | **32명** ⭐ |
| 큰 효과 | 0.80 | **13명** |

### 권장사항
- **최소:** 30-35명 (중간 효과크기 기준)
- **이상적:** 50-60명 (작은 효과도 탐지)
- **보수적:** 80-100명 (탈락률 20% + 하위분석)

**현재 N=6은 파일럿 수준. 본 실험은 최소 30명 이상 필요.**

## 주요 문제점

### 1. **검정력 (Power)**
- N=6은 중간 효과(d=0.5)를 탐지하기에 **15% power**만 보유
- d=1.2-1.5 이상의 매우 큰 효과만 탐지 가능
- **해결:** N≥30으로 증가

### 2. **그럴듯함 조작**
- 기저선(중립 조건)에서도 그럴듯함 효과 없음
- 참가자들이 비그럴듯함을 인지하지 못했을 가능성
- **해결:** 독립 pilot test로 사전 검증, 문장 단순화

### 3. **영역 구분**
- 여러 단어를 묶어 평균내면 효과가 희석될 수 있음
- **해결:** 단어별 분석, 특히 핵심 명사와 첫 spillover
